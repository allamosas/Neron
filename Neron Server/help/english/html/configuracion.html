Server Settings Nero

Port

The port is an interface through which the application can handle connections of the mobile application.
For this, the router configuration must be enabled. Before you enable ports you should make sure you have a private fixed IP address. If you do not, Redeszone offers a good guide.
The configuration of the router ports to enable changes depending on the brand. Portforward.com is a page they can help you open them properly. You should only enter the make and model and follow the steps.

FQDN

The public IP constantly changing, and if you're going to use the mobile application from a different network to the server should associate a FQDN.
NO-IP is a service that allows you to associate your computer with a FQDN so can connect to it although public IP changes. You should only install the application on your computer, choose the name of your FQDN and insert it in the mobile application. The downside of this is that you can not access this mode if the computer and the device They are in the same mobile network.
