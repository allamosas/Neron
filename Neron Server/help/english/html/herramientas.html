New file

With this option, you can add files and directories. It will open a wizard from which you can choose one or more items to add to the table. In the event already on the table will be ignored.

Also you can be added by dragging files to the table on it.



Deleting files

In case you have selected one or more items in the table this tool will be activated allowing remove them from the list. I dont know physically deleted from your computer.



Save list
