Paramètres du serveur Nero

Port

Le port est une interface par laquelle l'application peut gérer les connexions de l'application mobile.
Pour cela, la configuration du routeur doit être activé. Avant d'activer les ports, vous devez vous assurer que vous avez une adresse IP fixe privée. Si vous ne le faites pas, Redeszone offre un bon guide.
La configuration des ports de routeur pour permettre des modifications en fonction de la marque. Portforward.com est une page ils peuvent vous aider à ouvrir correctement. Vous ne devez entrer dans la marque et le modèle et suivez les étapes.

FQDN

L'IP publique en constante évolution, et si vous allez utiliser l'application mobile à partir d'un autre réseau au serveur devrait associer un FQDN.
NO-IP est un service qui vous permet d'associer votre ordinateur avec un FQDN afin peut se connecter à ce même si des changements IP publiques. Vous ne devez installer l'application sur votre ordinateur, choisissez le nom de votre FQDN et l'insérer dans l'application mobile. L'inconvénient de ceci est que vous ne pouvez pas accéder à ce mode si l'ordinateur et le périphérique Ils sont dans le même réseau mobile.
