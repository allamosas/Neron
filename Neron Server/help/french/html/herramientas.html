Outils de contrôle

Nouveau fichier

L'utilisation de cette option peut ajouter des fichiers et des répertoires. Il va ouvrir une Assistant à partir de laquelle vous pouvez choisir un ou plusieurs articles à ajouter à la table. Dans le cas déjà sur la table seront ignorées.

Vous pouvez également être ajouté en faisant glisser les fichiers à la table sur elle.


Suppression de fichiers

Dans le cas où vous avez sélectionné un ou plusieurs éléments dans le tableau cet outil sera activé permettant de les supprimer de la liste. Je ne sais pas physiquement supprimé de votre ordinateur.


Enregistrer la liste

Cette option sera utilisée à partir du moment où un changement est détecté dans la liste des fichiers et répertoires. Pour certaines actions seront Je dois enregistrer les paramètres avant de procéder
