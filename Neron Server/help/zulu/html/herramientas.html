ifayela New

Ukusebenzisa lolu khetho Ungengeza amafayela kanye amalayelo. Kuyoba ukuvula wizadi lapho ungase ukwazi khona ukukhetha into eyodwa noma ngaphezulu ukuze ungeze etafuleni. Esimweni kakade phezu kwetafula zizozitshwa.

Godu kungenziwa wanezela ngokuhudula amafayela etafuleni kuso.


Ukususa amafayela

Uma kwenzeka osikhethile into eyodwa noma ngaphezulu etafuleni leli thuluzi izoqaliswa ngokuvumela bawasuse ohlwini. akukho ngokomzimba zisusiwe ekhompyutheni yakho.


Londoloza uhlu

Lolu khetho luzosetshenziselwa kusukela esikhathini lapho ushintsho ibanjiwe ohlwini lwamafayela kanye Izincwajana. Ngoba izenzo ezithile kuyoba Ngimelwe ukulondoloza amasethingi ngaphambi kokuqhubeka
