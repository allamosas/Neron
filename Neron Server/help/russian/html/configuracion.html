Порт

Порт представляет собой интерфейс, с помощью которого приложение может обрабатывать подключения мобильного приложения.
Для этого, конфигурация маршрутизатора должен быть включен. Перед тем, как включить порты, которые вы должны убедиться, что у вас есть частный фиксированный IP-адрес. Если вы этого не сделаете, Redeszone предлагает хорошее руководство.
Конфигурация портов маршрутизатора для того, чтобы изменения в зависимости от марки. Portforward.com является страницей они могут помочь вам открыть их должным образом. Вы должны ввести только марку и модель и следуйте инструкциям.

полное доменное имя

Общественность IP постоянно меняется, и если вы собираетесь использовать мобильное приложение из другой сети на сервер должен ассоциировать полное доменное имя.
NO-IP это услуга, которая позволяет связать компьютер с доменным именем так может подключиться к нему, хотя изменения общественных IP. Вы должны установить только приложение на вашем компьютере, выберите имя вашего полного доменного имени и вставить его в мобильном приложении. Недостатком этого является то, что вы не можете получить доступ в этот режим, если компьютер и устройство Они находятся в той же сети мобильной связи.
