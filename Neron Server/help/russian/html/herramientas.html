Инструменты управления

Новый файл

С помощью этой опции можно добавлять файлы и каталоги. Он будет открыт Мастер, из которого вы можете выбрать один или несколько элементов, чтобы добавить к столу. В случае уже на столе будет проигнорирован.

Также можно добавить путем перетаскивания файлов в таблицу на него.



Удаление файлов

В случае, если вы выбрали один или несколько элементов в таблице этот инструмент будет активирован позволяющий удалить их из списка. нет физически удалены с вашего компьютера.



Сохранить список
