Настройки пользователя

Пункт, из которого вы можете настроить некоторые пользовательские настройки



Изменить пароль

Вы можете изменить пароль, используемый приложением и аутентификации доступа и аутентификации на мобильном устройстве для удаления и проверка соединения.

Для этого необходимо ввести текущий пароль с новым паролем, который будет повторяться и проверяется по соображениям безопасности.



Изменить язык

В этом меню вы можете изменить язык приложения либо из доступны. Некоторые языки содержат символы, которые не поддерживаются некоторыми вопросы реализации.



Изменить тему

В этом меню вы можете изменить тему приложения к любому из доступны. Некоторые темы не поддерживают определенные символы доступных языков приложение.



Уведомления

В этом разделе можно настроить все оповещения на рабочем столе, которые будет посылать приложение для пользователя.

  • Это позволяет приложению уведомляет пользователя всякий раз, когда вы входите в приложение сколько файлов или каталогов, хранящиеся в списке удаления не Я мог бы быть найден.
  • Это позволяет приложению проверять каждый раз, если файлов и директорий присутствует в таблице, были перемещены или удалены.
  • Разрешить приложение отправляет процесс уведомления на рабочем столе устранение элементов таблицы с помощью мобильного приложения
  • Это позволяет приложению известить результат процесса устранение элементов таблицы с помощью мобильного приложения
  • Это позволяет приложению уведомлять пользователя информацией устройство, которое вы пытаетесь подключиться к приложению в то время. Эта информация должна содержать IP-адрес устройства, и если пароль, используемый соединение было правильным или неправильным.
