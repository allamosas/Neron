Port

La porta è un'interfaccia attraverso la quale l'applicazione può gestire le connessioni di applicazioni mobile.
Per questo, la configurazione del router deve essere abilitato. Prima di attivare le porte si dovrebbe assicurarsi di avere un indirizzo IP fisso privato. Se non lo fai, Redeszone offre una buona guida.
La configurazione delle porte del router per abilitare le modifiche a seconda della marca. Portforward.com è una pagina possono aiutare aprirli correttamente. Si deve inserire solo la marca e il modello e seguire le istruzioni.

FQDN

Il IP pubblico in continua evoluzione, e se avete intenzione di utilizzare l'applicazione mobile da una rete diversa per il server dovrebbe associare un nome di dominio completo.
NO-IP è un servizio che permette di associare il computer con un nome di dominio completo così in grado di connettersi ad esso, anche se le modifiche IP pubblici. Si consiglia di installare solo l'applicazione sul computer, scegliere il nome del tuo nome di dominio completo e inserirla nell'applicazione mobile. L'aspetto negativo di questo è che non è possibile accedere a questa modalità se il computer e il dispositivo Essi sono nella stessa rete mobile.
