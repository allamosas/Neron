Opzioni utente

Il paragrafo da cui è possibile configurare alcune preferenze degli utenti


Modifica password

È possibile modificare la password utilizzata per l'applicazione sia l'autenticazione l'accesso e l'autenticazione sul dispositivo mobile per la rimozione e il test di connessione.

Per fare questo è necessario immettere la password attuale con la nuova password, che si ripeterà e controllato per motivi di sicurezza.


Cambia lingua

Da questo menu è possibile cambiare la lingua dell'applicazione o del disponibili. Alcuni linguaggi contengono caratteri che non sono supportati da alcuni problemi di implementazione.


Cambia tema

Da questo menu è possibile modificare l'oggetto della domanda ad uno qualsiasi dei disponibili. Alcuni temi non supportano alcuni caratteri delle lingue disponibili applicazione.


Notifiche

Questa sezione permette di impostare tutte le notifiche desktop che invierà il applicazione per l'utente.

  • Permette l'applicazione avvisa l'utente ogni volta che si entra l'applicazione il numero di file o le directory presenti nell'elenco di eliminazione non avere Mi è stato trovato.
  • consente all'applicazione di controllare ogni volta se i file e le directory presente nella tabella sono stati spostati o eliminati.
  • Consentire l'applicazione invia un processo di notifica sul desktop eliminazione degli elementi della tabella utilizzando l'applicazione mobile
  • consente all'applicazione di comunicare il risultato del processo di eliminazione degli elementi della tabella utilizzando l'applicazione mobile
  • consente all'applicazione di informare l'utente con le informazioni il dispositivo si sta tentando di connettersi all'applicazione in quel momento. Tali informazioni devono contenere l'indirizzo IP del dispositivo e se la password utilizzata il collegamento è stato giusto o sbagliato.
