Strumenti di controllo

New file

Utilizzando questa opzione è possibile aggiungere file e directory. Si aprirà una guidata da cui è possibile scegliere uno o più elementi da aggiungere al tavolo. Nel caso già sul tavolo verrà ignorata.

Inoltre è possibile essere aggiunto trascinando i file al tavolo su di esso.


L'eliminazione di file

Nel caso in cui avete selezionato uno o più elementi nella tabella questo strumento sarà attivato permettendo rimuoverli dalla lista. no cancellato fisicamente dal computer.


Salva lista

Questa opzione verrà utilizzato dal momento in cui viene rilevato un cambiamento nella lista di file e directory. Per alcune azioni saranno Devo salvare le impostazioni prima di procedere
