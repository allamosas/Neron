Танзимотҳо Server Нерон

Port

Дар бандари интерфейси, ки тавассути он ариза метавонад алоқаҳои аз барномаҳои мобилӣ ҳал аст.
Барои ин, конфигуратсияи роутер бояд фаъол шавад. Пеш аз он ки ту ба бандарҳои фаъол шумо бояд боварӣ ҳосил кунед, суроғаи IP муайян хусусӣ. Агар шумо корро надорад, Redeszone пешниҳод роҳнамои хуб.
Дар конфигуратсияи бандарҳои роутер ба имконият таѓйироти вобаста ба бренди. Portforward.com саҳифаи аст, онҳо метавонанд ба шумо кӯмак ба онҳо дуруст кушояд. Шумо танҳо бояд ворид задан ва модели ва қадамҳои пайравӣ.

FQDN

Дар IP ҷамъиятӣ мунтазам тағйир меёбад, ва агар шумо ба зудӣ истифодаи барномаи мобилӣ аз шабакаи гуногун ба сервер бояд шарики як FQDN.
НЕ-IP хизматрасоние мебошад, ки ба шумо имкон медиҳад, то ки шарики компютери шумо бо як FQDN, то аз метавонад ба он ҳарчанд тағйироти IP ҷамъиятӣ пайваст шавед. Шумо танҳо бояд ба насби барномаи дар компютери шумо, номи FQDN худро интихоб ва аз ҷойгиркунии он дар ариза мобилӣ. Бо нуқсони ин аст, ки шумо метавонед ин ҳолатро агар дастрас нест, компютер ва дастгоҳ Онҳо дар шабакаи мобилӣ ҳамон аст.
