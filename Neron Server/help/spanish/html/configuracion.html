Configuración del servidor Nerón

Puerto

El puerto es una interfaz mediante la que la aplicación puede atender las conexiones de la aplicación móvil.
Para esto, en la configuración del router debe estar habilitado. Antes de habilitar los puertos deberías asegurarte de tener una dirección IP privada fija. Si no la tienes, Redeszone propone una buena guía.
La configuración del router para habilitar puertos cambia en función de la marca. Portforward.com es una página en la que te pueden ayudar a abrirlos correctamente. Sólo deberías introducir la marca y el modelo y seguir los pasos indicados.

FQDN

Las IP públicas cambian constantemente, y si vas a usar la aplicación móvil desde otra red distinta a la del servidor convendría asociarle un FQDN.
NO-IP es un servicio que te permite asociar tu equipo con un FQDN para así poder conectar con él aunque la IP pública cambie. Sólo deberías instalar la aplicación en el equipo, elegir el nombre de tu FQDN e introducirlo en la aplicación móvil. La desventaja de esto es que no se puede acceder de este modo si el equipo y el dispositivo móvil están en la misma red.
