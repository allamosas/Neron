Opciones de usuario

Apartado desde el cual se podrán configurar ciertas preferencias del usuario



Cambio de contraseña

Se permite cambiar la contraseña usada por la aplicación tanto en la autenticación en el acceso como la autenticación en el dispositivo móvil para la eliminación y la prueba de conexión.

Para ello se tiene que introducir la contraseña actual junto a la nueva contraseña, la cual será repetida y comprobada por motivos de seguridad.



Cambio de idioma

Desde este menú se puede cambiar el idioma de la aplicación uno cualquiera de los disponibles. Algunos idiomas contienen caracteres que no son soportados por ciertos temas de la aplicación.



Cambio de tema

Desde este menú se puede cambiar el tema de la aplicación a uno cualquiera de los disponibles. Algunos temas no soportan ciertos caracteres de los idiomas disponibles de la aplicación.



Notificaciones

Este apartado permite configurar todas las notificaciones de escritorio que enviará la aplicación al usuario.

  • Se permite que la aplicación notifique al usuario cada vez que entre en la aplicación sobre cuántos ficheros o directorios guardados en la lista de eliminación no han podido ser encontrados.
  • Se permite que la aplicación compruebe cada cierto tiempo si los ficheros y directorios presentes en la tabla han sido movidos o eliminados.
  • Se permite que la aplicación envíe una notificación de escritorio del proceso de eliminación de los elementos de la tabla mediante el aplicativo móvil
  • Se permite que la aplicación envíe una notificación con el resultado del proceso de eliminación de los elementos de la tabla mediante el aplicativo móvil
  • Se permite que la aplicación envíe una notificación al usuario con la información del dispositivo que esté tratando de conectarse a la aplicación en ese momento. Ésta información contendrá la dirección IP del dispositivo y si la contraseña usada en la conexión ha sido correcta o incorrecta.


Configuración Avanzada

Desde este apartado se podrá especificar el puerto de escucha de la aplicación

También se podrá establecer el número de intentos que dispondrá la aplicación móvil para conectarse al servidor y el tiempo que deberá esperar para volver a intentarlo una vez haya sobrepasado el límite.
