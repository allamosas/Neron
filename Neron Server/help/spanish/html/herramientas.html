Control de Herramientas

Nuevo fichero

Mediante esta opción se pueden añadir ficheros y directorios. Se abrirá un asistente desde el que se podrán elegir uno o varios elementos para añadirlos a la tabla. En el caso de que ya se encuentren en la tabla serán ignorados.

También se pueden añadir ficheros a la tabla arrastrándolos sobre ella.



Eliminación de ficheros

En el caso de que hayan sido seleccionados uno o más elementos de la tabla se activará esta herramienta y permitirá eliminarlos de la lista. No se eliminarán físicamente del equipo.



Guardar lista

Esta opción será utilizable desde el momento en el que se detecte un cambio en la lista de ficheros y directorios. Para determinadas acciones estará obligado guardar la configuración antes de proseguir
